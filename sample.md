### コード

インラインコード: `console.log('Hello World')`

```javascript
// JavaScriptのコードブロック
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```

```python
# Pythonのコードブロック
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```

## 検索・置換機能のテスト

この文章には「テスト」という単語が複数回出現します。検索・置換機能を使って「テスト」を「サンプル」に置き換えることができます。

### 正規表現のテスト

以下のパターンを検索できます：

- `\d+` - 数字の連続
- `[A-Z]+` - 大文字の連続
- `\b\w+\b` - 単語境界
