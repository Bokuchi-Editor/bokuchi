# モンキーテストケースリスト

自動テストではカバーしきれないUI統合動作を手動で確認するためのテストケース。
本来はE2Eで実現したいが、今はTauriの対応状況的に不可能なので手動で行う。

## 1. タブ操作

| #    | テストケース                    | 操作手順                                       | 期待結果                                                |
| ---- | ------------------------------- | ---------------------------------------------- | ------------------------------------------------------- |
| M-01 | 新規タブ作成                    | `Ctrl+N` を押す                                | 新しいタブが開き、エディタにフォーカスが当たる          |
| M-02 | 複数タブ切り替え                | 3つ以上タブを開き、各タブをクリック            | 正しい内容が表示され、エディタにフォーカスが移る        |
| M-03 | Ctrl+Tab タブ切り替え           | `Ctrl+Tab` で次タブ、`Ctrl+Shift+Tab` で前タブ | 順序通りに切り替わり、末尾→先頭のラップも動作する       |
